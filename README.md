# Образовательный портал

Платформа для создания и изучения курсов по дизайну и игровой разработке.

## Функциональность

- ✅ Регистрация и авторизация пользователей
- ✅ Создание курсов (любой пользователь может быть автором)
- ✅ Редактирование и удаление собственных курсов
- ✅ Запись на курсы других авторов
- ✅ Личный кабинет с прогрессом обучения
- ✅ Административная панель
